# 🚀 План развития проекта: Дорожная карта улучшений

Этот файл содержит идеи для расширения симуляции «Непредсказуемого проекта». Функции можно внедрять постепенно, наблюдая за тем, как они меняют динамику цифрового мира.

## 1. 🧪 Химическая коммуникация (Феромоны)
Организмы смогут общаться, оставляя «метки» прямо на игровом поле.
- [x] **Эмиссия**: Добавить команду генома для выделения феромона (разных типов: например, «Еда» или «Тревога»).
- [x] **Диффузия и распад**: Реализовать слой карты, где запах постепенно расплывается и исчезает со временем.
- [x] **Сенсоры**: Новые команды генома, которые считывают уровень феромона в соседних клетках и корректируют направление движения.

## 2. 🦠 Система вирусов (Инфекционный код)
Введение чистого информационного паразитизма.
- [x] **Вирусные цепочки**: Создать сущности, состоящие только из короткого набора байт (без энергии).
- [x] **Инфицирование**: При контакте вирус внедряет свой код в геном хозяина.
- [x] **Репликация**: Зараженный организм при выполнении вирусного участка тратит энергию на создание копий вируса.
- [x] **Мутации вирусов**: Вирусы тоже должны эволюционировать вместе с хозяевами.

## 3. 👁 Улучшенное восприятие (Зрение и Сканеры)
Переход от слепого перемещения к тактическому поведению.
- [x] **Дальний поиск**: Поиск ресурсов в радиусе 5 клеток (Зрение).
- [x] **Детектор угроз**: Обнаружение хищников и паразитов в радиусе 4 клеток (Сканер).
- [x] **Социальный радар**: Поиск сородичей своего клана для координации (Радар).

## 4. 📜 Генератор «Лора» и Именованных Кланов
Сделать статистику более человекочитаемой и интересной.
- [x] **Семантические имена**: Автоматическое присвоение имен видам на основе их доминирующих признаков (например, *«Кровавые Спринтеры»* для красных агрессивных особей).
- [x] **История мира**: Логирование великих империй (видов, которые доминировали дольше всего) и причин их краха.

## 5. ⛈ Динамические катаклизмы
Внешние факторы, проверяющие популяцию на прочность.
- [ ] **Эпидемии**: Глобальное падение шанса на размножение на определенное время.
- [ ] **Метеоритные удары**: Мгновенная очистка области с последующим выбросом огромного количества ресурсов.
- [ ] **Засуха/Наводнение**: Временное изменение биологической ценности биомов.

## 6. 🖐 Режим «Рука Бога» (Интерактивность)
Инструменты прямого вмешательства пользователя.
- [ ] **Кисть ресурсов**: Возможность за секунды создать «райский сад» или «пустыню».
- [ ] **Лабораторный образец**: Инструмент для захвата особи, просмотра её полного кода и ручного его редактирования.
- [ ] **Стихийный спавн**: Кнопка для ручного занесения мутантного или вирусного кода в популяцию.

---
*Каждая из этих функций превратит симуляцию из простого наблюдения в захватывающее исследование эмерджентной сложности.*
