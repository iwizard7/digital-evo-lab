# Непредсказуемый проект | Digital Life Lab

**Непредсказуемый проект** — это интерактивная среда симуляции искусственной жизни (Artificial Life), где цифровые организмы эволюционируют, соревнуются за ресурсы и адаптируются к меняющимся условиям среды.

## 🧬 О проекте

Проект представляет собой «песочницу», в которой биомы населены байтовыми организмами. Каждый организм обладает собственным геномом, который определяет его поведение, метаболизм и стратегию выживания. Симуляция позволяет наблюдать за возникновением сложных биологических явлений, таких как паразитизм, симбиоз, массовые вымирания и горизонтальный перенос генов.

## 🚀 Основные возможности

- **Эволюционный движок**: Реалистичная модель мутаций и естественного отбора.
- **Байт-код поведение**: Организмы исполняют свой геном как программу.
- **Разнообразие стратегий**:
    - `Normal`: Обычное выживание и сбор ресурсов.
    - `Parasite`: Кража энергии у других организмов.
    - `Cooperative`: Альтруистичное поведение и поддержка сородичей.
- **Семантические имена кланов**: Группы организмов теперь получают уникальные названия (например, «Яростные Пожиратели» или «Тихие Странники») на основе их поведения и генома.
- **Динамическая среда и Феромоны**: 
    - Система биомов с сезонными циклами.
    - **Феромоны**: Организмы могут оставлять химические следы (фиолетовое свечение).
- **Улучшенное восприятие**:
    - **Зрение ресурсов**: Поиск богатых едой участков в широком радиусе.
    - **Детектор угроз**: Автоматическое бегство от агрессивных мутантов и паразитов.
    - **Социальный радар**: Способность находить своих собратьев для выживания в группах.
- **Глубокая аналитика и Хроники**:
    - **Хроники мира**: Отслеживание великих империй прошлого.
    - Анализ причин смертности (Голод, Старость, Атаки, Дефицит памяти, Крах).
- **Динамические катаклизмы**:
    - **Метеориты**: Случайные удары, уничтожающие жизнь в радиусе, но удобряющие почву.
    - **Эпидемии**: Массовые болезни, снижающие плодовитость и повышающие износ.
    - **Засухи**: Периоды жесткого дефицита ресурсов по всему миру.
- **Высокая производительность**: Вычисления вынесены в фоновый поток через **Web Workers**, что обеспечивает плавную отрисовку при больших популяциях.

## 🛠 Технологический стек

- **Core**: JavaScript (ES6+), Web Workers API.
- **Frontend**: HTML5 Canvas, Vanilla CSS.
- **Fonts**: Google Fonts (Manrope, Space Grotesk).
- **Architecture**: Модульная структура с разделением логики симуляции (`sim-core.mjs`), фоновых вычислений (`sim-worker.mjs`) и визуального интерфейса (`app.mjs`).

## 📋 Инструкция по запуску

Для запуска проекта достаточно любого современного браузера. Так как проект использует ES-модули, его необходимо открывать через локальный сервер (например, Live Server во VS Code или любой другой статический сервер).

1. Скачайте файлы проекта.
2. Запустите локальный сервер в корневой директории (например, через расширение VS Code `Live Server` или командой `npx http-server`).
3. Откройте `index.html` в браузере.

## 🛑 Завершение работы

Для корректного выхода из симуляции рекомендуется:
1. **Сохранить состояние**: Нажмите кнопку «Сохранить» в интерфейсе, чтобы скачать JSON-файл с текущим прогрессом.
2. **Поставить на паузу**: Нажмите «Пауза», чтобы остановить вычисления в фоновом потоке.
3. **Остановить сервер**: Если сервер запущен в терминале, нажмите `Ctrl + C`.

## 🎮 Управление симуляцией

- **Панель управления**:
    - `Пауза/Запуск`: Остановка времени.
    - `Шаг`: Продвижение симуляции на один тик.
    - `Мутационный всплеск`: Резкое повышение вероятности мутаций для стимуляции эволюционных изменений.
    - `Сохранение/Загрузка`: Экспорт текущего состояния мира в JSON.
- **Параметры и пресеты**:
    - Выбирайте готовые сценарии («Мир паразитов», «Сбалансированный» и др.) или настраивайте параметры (шанс мутаций, плотность пищи, лимит памяти) вручную.
- **Визуализация**:
    - Клавиша `H`: Переключение отображения теплокарты ресурсов (биомов).
    - Нижние графики: Отслеживание истории популяции и родственных связей (Флогененетика).

## 📁 Структура файлов

- `index.html` — Основная разметка интерфейса и холсты (Canvas) для графиков.
- `styles.css` — Современный адаптивный дизайн в темных тонах.
- `app.mjs` — Точка входа, управление UI и связь с Worker-ом.
- `sim-core.mjs` — Ядро симуляции: логика жизни, мутаций, коллизий и биомов.
- `sim-worker.mjs` — Воркер для выполнения шагов симуляции в отдельном потоке.
- `lab/` — Директория для дополнительных экспериментальных модулей.
